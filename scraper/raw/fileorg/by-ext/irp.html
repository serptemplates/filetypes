What are IRP files and how to open them

Are you having problems opening an IRP file or are you simply curious about its contents? We're here to explain the properties of these files and provide you with software that can open or handle your IRP files.

What is an IRP file?

An .IRP file is an InfraRecorder Project file.

We have not yet analyzed in detail what these files contain and what they are used for. We're working on it.

How to open IRP files

Important: Different programs may use files with the IRP file extension for different purposes, so unless you are sure which format your IRP file is, you may need to try a few different programs.
