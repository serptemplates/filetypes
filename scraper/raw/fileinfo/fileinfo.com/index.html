File Type of the Day

Today • November 1

.DCR

Shockwave Media File

A DCR file is an interactive media file saved in the Shockwave format for distribution on the web. It is often used for creating web games and other highly interactive content.

October 31

.CGI

Common Gateway Interface Script

A CGI file is a Common Gateway Interface (CGI) script executed by a web server to create dynamic pages. It is typically written in Perl or C programming languages.

October 30

.ARF

WebEx Advanced Recording File

An ARF file contains a recording of an online meeting downloaded from WebEx. It is recorded using Network-Based Recording (NBR) and stores the video data, a table of contents, an attendee list, and other information on separate panels.
